Teste para criação de uma página de listagem de produtos, que irá conter:

- Menu (Mobile)
- Listagem de produtos (busca da listagem através de XHR)
- Paginação a cada 20 produtos
- Um "filtro" de ordenação
  - Maior valor
  - Menor valor
  - Nome A-Z
  - Nome Z-A
